# ✉️关于文档的说明

### ✨介绍

该开源文档涉及的主要内容包括：Redis、MySQL、Java、主流框架、计算机网络、操作系统等面试必备知识。相信你能在这份文档学到不少专业技能、面试重点。

### 📖我为什么要做这个开源项目

目前网络上分享的各种面经包含的知识点是非常零散分散的，而面试官面试真正想考量你的其实是体系的知识。

例如绝大多数大厂面试官问上一个问题，是为下一个问题作准备，问完一个接着问：还有吗？还有吗？然后呢？其实这么问的主要目的就是考核你对整个体系知识的掌握程度。

靠记住零散的面试题对我们的技术提升并不大，也回答不了面试官对某一个模块刁钻的考核。

这个开源文档把每个模块系统的知识都编写为面试官刁钻考核的形式，相信能帮助到大家学习Java系统核心知识，顺利通关面试、拿到理想Offer。
